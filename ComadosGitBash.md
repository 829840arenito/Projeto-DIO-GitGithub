# Comandos GitBash #

### Git init - ###

Inicia o repositório.

### Git add - ###

Move arquivos e inicia o versionamento.

### Git commit - ###

Cria o commit.

### Git status - ###

Indica a situação do arquivo.

### Git clone - ###

Copia o repositório.

### Git push - ###

Envia do servidor local para o Git.
